# Pod控制器

### StatefulSet 

+ 有稳定网络标志，Pod重新调度后PodName和HostName不变，是基于Headless Service来实现

* 有序部署，有序扩展，有序收缩，有序删除（实现Running Man和ready状态）

+ Horizontal Pod Autoscaling HPA 高峰和低谷实现Pod自动调整数。

~~~yaml
## ReplicaSet 类型RS
apiVersion: apps/v1
kind: ReplicaSet
metadata:
  name: frontend
spec:
  replicas: 3
  selector:
    matchLabels:
      tier: frontend
  template:
    metadata:
      labels:
        tier: frontend
    spec:
      containers:
      - name: php-redis
        image: www.harbor.mobi/library/wangyanglinux/myapp:v1
        env:
        - name: GET_HOSTS_FROM
          value: dns
        ports:
        - containerPort: 80
~~~

### 部署一个简单的 Nginx 应用  



~~~yaml
## Deployment 类型Deployment
## Deployment，添加selector,matchLabels
apiVersion: apps/v1
kind: Deployment
metadata:
  name: nginx-deployment
spec:
  selector:
    matchLabels:
      app: nginx
  replicas: 3
  template:
    metadata:
      labels:
        app: nginx
    spec:
      containers:
      - name: nginx-one
        image: www.harbor.mobi/library/wangyanglinux/myapp:v1
        ports:
        - containerPort: 80

~~~
